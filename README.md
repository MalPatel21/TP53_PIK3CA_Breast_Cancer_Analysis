# TP53_PIK3CA_Breast_Cancer_Analysis

Breast cancer is a heterogeneous disease characterised by substantial biological and clinical variation between tumours. This project investigates the relationship between two of the most frequently mutated breast cancer genes, **TP53** and **PIK3CA**, across the **Luminal A (LumA)** and **Luminal B (LumB)** molecular subtypes defined by the PAM50 classification system.

Using data from the **Molecular Taxonomy of Breast Cancer International Consortium (METABRIC)**, this analysis explores mutation patterns in TP53 and PIK3CA and examines their association with **overall survival**. The aim is to better understand how genomic alterations interact with molecular subtype and patient prognosis in breast cancer.
